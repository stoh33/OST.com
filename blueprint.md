# OST.com Project Blueprint

## Project Overview
A web-based lotto number recommender with a personal touch. Features a modern, interactive UI built with framework-less HTML, CSS, and JS.

## Current Project State
- `index.html`: Main entry point with UI for lotto generation and a special message.
- `main.js`: Logic for generating 5 sets of random lotto numbers.
- `style.css`: Modern, responsive styling with premium visual effects (gradients, shadows, interactivity).
